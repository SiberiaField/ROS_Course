===
Название: rmdir
Описание: Удаление пустых директорий
Синтаксис: rmdir [опции] директория
Пример: rmdir -p a/b/c
Примечание: Для удаления не пустых директорий можно использовать "rm -r"

===
Название: wc
Описание: Считает кол-во строк, слов, байт в файлах
Синтаксис: wc [опции] [файлы]
Пример: wc -l file_1.txt file_2.txt
Примечание: Выводит результат для каждого файла по-отдельности и суммарно

===
Название: man
Описание: Форматирует и выводит онлайн страницы мануала
Синтаксис: man [опции] имя_команды
Пример: man cat
Примечание: Сохраняет форматированные страницы для экономии времени в будущем

===
Название: touch
Описание: Изменяет временные штампы файла
Синтаксис: touch [опции] файл
Пример: touch -m file.txt
Примечание: Чаще используется чтобы создать новый пустой файл (touch new_file)

===
Название: ps
Описание: Выводит информацию о текущих процессах
Синтаксис: ps [опции]
Пример: ps
Примечание: Стандратно(без флагов) выводит информацию о процессах только в текущем терминале

===
Название: history
Описание: Выводит список недавно выполненных команд
Синтаксис: history [опции] файл или history -ps arg [аргументы]
Пример: history
Примечание: Проблематично найти мануал именно по команде, а не по GNU библиотеке

===
Название: shred
Описание: Переписывает файл, скрывая его содержимое, и выборочно удаляет его
Синтаксис: shred [опции] файл [...]
Пример: shred -v -f file.txt
Примечание: -

===
Название: tail
Описание: Выводит последние строки файла
Синтаксис: tail [опции] [файл]
Пример: tail -n=15 file.txt
Примечание: Если файл не указан, что читает стандратный ввод. Есть команда head, которая читает первые строки файла

===
Название: unmae
Описание: Сообщает информацию о данном компьютере и операционной системе
Синтаксис: uname [опции]
Пример: uname -a
Примечание: -

===
Название: find
Описание: Найти файлы в иерархии директорий
Синтаксис: find [опции] [пути] [выражение]
Пример: find ./ -name "test.txt"
Примечание: Если путь не указан, то используется текущая директория
